### What Just Happened? (Under the Hood)

When you call `generate_exceptional_preview()`, the system:

1. **Contextual Intelligence** (Layer 6) analyzes the URL → classifies industry → detects audience → recommends design approach
2. **Visual Hierarchy** (Layer 1) calculates dominance scores → assigns hierarchy levels (Hero, Primary, Secondary) → applies optical sizing
3. **Dynamic Composition** (Layer 5) selects optimal grid system → calculates layout zones → balances visual weight
4. **Premium Typography** (Layer 3) selects font pairing based on brand personality → applies optical tracking
5. **Generates base image** with contextually-appropriate colors, layout, and typography
6. **Texture & Materials** (Layer 4) applies subtle film grain or paper texture → blends with overlay mode
7. **Depth & Shadows** (Layer 2) applies 3-layer shadow composition → adds subtle vignette for depth
8. **Quality Assurance** (Layer 7) validates WCAG AAA compliance → applies auto-polish → returns grade (A+, A, B+, etc.)

**Result**: Professional, designer-quality preview optimized for the specific industry and audience.

## 🎨 The 7 Layers Explained (TL;DR)

### 1. Visual Hierarchy Engine
**What**: Professional multi-level hierarchy  
**How**: Dominance scoring → level assignment → optical sizing  
**Example**: Hero headline (96px), primary subhead (48px), secondary description (32px)

### 2. Depth & Shadow System
**What**: Material Design 3.0 shadows  
**How**: 3-layer composition (ambient, penumbra, umbra)  
**Example**: Raised card gets: 0 2px 4px rgba(0,0,0,0.06), 0 2px 8px rgba(0,0,0,0.12), 0 2px 2px rgba(0,0,0,0.08)

### 3. Premium Typography Engine
**What**: 50+ professional font pairings  
**How**: Personality-based selection → optical kerning  
**Example**: Fintech → "Corporate Authority" (Montserrat Bold + Inter Regular)

### 4. Texture & Materials Engine
**What**: Procedural textures and patterns  
**How**: NumPy-based generation → blend modes  
**Example**: Subtle film grain (3% intensity) adds premium feel

### 5. Dynamic Composition Engine
**What**: Intelligent grid systems  
**How**: Content analysis → grid selection → zone calculation  
**Example**: Headline-focused content → Swiss grid with centered hero

### 6. Contextual Intelligence Engine
**What**: Industry-aware design  
**How**: URL classification → profile matching → color/style selection  
**Example**: stripe.com → Fintech → Blue/Green palette + Authoritative typography

### 7. Quality Assurance Engine
**What**: Automated validation and polish  
**How**: WCAG checks → balance scoring → auto-enhance  
**Example**: Low contrast detected → auto-adjusted to 7:1 ratio (AAA)
